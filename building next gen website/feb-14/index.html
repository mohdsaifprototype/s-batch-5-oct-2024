Simple selector Part-2

We have many types of advance selectors such as

  1. Pseudo selector

    Those selector that applies styling to the element via css only. To identify a pseudo you will have to check :pseudo-selector or ::pseudo-selector sign.

    Pseudo selector is further divided into two categories name:

    1. Pseudo class

      A pseudo class is mentioned with a single colon sign i.e., :pseudo-class

    2. Pseudo element

      A pseudo element is mentioned with double colon sign i.e., ::pseudo-element
